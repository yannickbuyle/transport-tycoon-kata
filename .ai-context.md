I # AI Agent Context - Transport Tycoon Kata

## Project Overview
Parallel logistics simulation exercise where containers are delivered from a Factory to warehouses using trucks and ship.

## Core Domain

### Map & Routes
- Factory → Port: 1 hour
- Port → A (ship only): 4 hours (6h in Exercise 1.2)
- Factory → B: 5 hours

### Transport Fleet
- **2 Trucks**: Start at Factory, carry 1 container each
- **1 Ship**: Starts at Port, carries 1 container (4 containers in Exercise 1.2)

### Core Rules
- FIFO container selection from locations
- All transport moves in **parallel** (not sequential)
- Instant loading/unloading (Exercise 1.1)
- Travel time applies both ways

## Exercise Progression

### Exercise 1.1
Calculate total delivery time for container sequences.

**Test Cases**:
- A → 5h
- AB → 5h
- BB → 5h
- ABB → 7h
- AABABBAB → 29h
- ABBBABAAABBB → ?

### Exercise 1.2
Add event logging and new ship rules.

**New Ship Rules**:
- Capacity: 4 containers
- Load time: 1 hour for all cargo
- Unload time: 1 hour for all cargo
- Travel time: 6 hours each direction

**Event Logging**: JSON format with DEPART/ARRIVE/LOAD/UNLOAD events
- Used for trace visualization in Chrome Trace Viewer
- Enables cross-language solution comparison

## Known Issues
- Non-deterministic results possible due to tie-breaking when events occur simultaneously
- This is acknowledged in the exercise design
